One Piece
Website

2025  ·  HTML / CSS / JavaScript  ·  netnet.studio

A multi-page informational website exploring the world of One Piece — built as a school assignment and personal passion project. The site takes users through a narrative journey from the grand homepage, through character pages for the Straw Hat crew, to an exploration of Devil Fruits and the power systems that shape the world.

Key Features

1440px Design Framework

1200px content container for balanced, centered layouts across all four pages.

Scroll-Triggered Animations

JavaScript Intersection Observer API powers fade-in reveals as users scroll.

Magazine-Style Layout

Absolute positioning creates dynamic, manga-inspired layouts with overlapping images and text.

CSS Keyframe Animations

Floating logo, glowing Joyboy illustration, and smooth element transitions throughout.

Custom Typography

"Protest Revolution" headers paired with "Annie Use Your Telescope" body text for a handcrafted feel.

Multi-Page Navigation

Four interconnected pages — Home, Crew, Luffy, and Devil Fruits — with consistent navigation.

One Piece © Eiichiro Oda/Shueisha. Non-commercial educational fan project, not intended for sale.

      const observer = new IntersectionObserver((entries) => {
        entries.forEach(entry => {
          if (entry.isIntersecting) {
            entry.target.style.opacity = '1';
            entry.target.style.transform = 'translateY(0)';
          }
        });

      fadeElements.forEach(el => {

        el.style.transform = 'translateY(30px)';

        observer.observe(el);
