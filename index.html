One Piece Website

2025 • Website • netnet: HTML/CSS/Java

A multi-page informational website exploring the world of One Piece, created as a school assignment and personal passion project. This non-commercial, educational tribute combines academic web development requirements with genuine enthusiasm for Eiichiro Oda's manga series. The site takes users through a narrative journey—from the grand introduction on the homepage, through character-focused pages celebrating the Straw Hat crew and Luffy's transformation, to an exploration of Devil Fruits and the power systems that shape the world. As both a technical exercise and a fan tribute, the project allowed me to engage deeply with themes of freedom, friendship, and challenging unjust systems while practicing advanced layout techniques and creating a visually engaging, magazine-style reading experience.

Process

Key Features

  • 1440px design framework with 1200px content container for balanced, centered layouts
  • Scroll-triggered fade-in animations using JavaScript Intersection Observer API
  • Absolute positioning for dynamic, manga-inspired magazine layouts with overlapping images and text
  • CSS keyframe animations (floating logo, smooth transitions)
  • Custom typography combining "Protest Revolution" headers with "Annie Use Your Telescope" body text
  • Deep crimson and white color palette inspired by manga ink work and pirate aesthetics
  • Multi-page navigation system connecting four distinct pages
  • Responsive image positioning maintaining precise visual relationships
